Notes when formatting your data:

  • The first column of the -omics data file should be the sample names. And, the second column should be the sample meta-data. (i.e. Is cell derived from epithelial source?)
  • If there is no sample meta-data for the -omics datasets, please have all zero instead for the second column.
  • Two -omics data should be uploaded, and the first two columns (sample names and sample meta-data) from two -omics should be the same.
  • If the sample name is unknown, list the numbers instead.
  • Sample should be in the rows, and the feature should be in the columns.
  • Both sample and feature names must be unique and consist of a combination of common English letters, underscores and numbers for naming purpose. Latin/Greek letters are not supported.
  • Using empty for missing values are NOT allowed. (Replace all the missing values with NA)
